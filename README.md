staqlt AI Outreach Engine

This is a functional scaffold for a multi-channel AI outreach system.

Core Features:

AI Brain (OpenAI GPT-4): Generates context-aware, hyper-personalized messaging for leads.

Multi-Channel Bridge: Support for LinkedIn, WhatsApp, Email, and Twitter (scaffolded).

Channel-Aware Logic: Dynamically adjusts the tone and length of the message based on the platform.

Setup:

Run npm install.

Create a .env file with your OPENAI_API_KEY.

Run npm start to execute the mock campaign.
